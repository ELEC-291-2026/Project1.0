# Reflow Oven Controller

## Hardware Overview
* Microcontroller: DE10-Lite/DE1-SoC

## Tasks
- Start/Stop button
- Display Right Stuff on the right displays
- Speaker with 1 beep when in new state and 5 beeps when done

## Extra functionality 
  - Abort automatically if specific conditions arent met
  - Show a warning message whenever there is a very large deviation in temperatures
  - Change the oven activation so that we have to hold it down for 3 seconds to start the process
  - Play different noises on a speaker for which state the oven is in
  - Display what state the oven is in on the bottom right of the LCD
  - Add a progress indicator and progress bar
  - A countdown timer that shows how many seconds are left in the state
  - Short song, once done, instead of 5 beeps
